& 在数组中的应用

以下面这个数组为例，我们先来明确一些概念

int a[10];

a与&a[0]的关系

  • 数组名a是数组0号元素的地址,也就是存在以下等价关系

    a 等价于 &a[0]

  • &a[0] 表示对一个int型变量取地址，a[0]是一个int变量， 一个int变量的地址类型是 int* ,所以 &a[0] 为 int* 类型

  • 于是有 a 等价于 &a[0]，且二者均为 int* 类型

a与&a的关系

  • &a 和 a 在数值上相同，但意义不同！

  • &a 代表数组地址，类型为 int(*)[10]

  • a 代表 数组0号元素的地址，类型为 int*

  • 于是指向数组的指针可以定义为：

    int(*p_arr)[10] = &a;

  • 指向数组成员的指针可以定义为：

    int *p = a; 或 int *p = &a[0];

a+i 中的隐式转换

我们经常在程序中看到类似这样的用法

printf("%d\n", *(a+i));

a 为int类型，i为 int类型，a+i的结果是int类型，加几就越过几个sizeof(int)，所以

a+i 或 &a[0]+i = 被取址变量本身的地址 + i * sizeof(被取址的变量的类型)

这个关系很重要，有的时候我们为使代码简洁，会直接对变量取址，进行上述操作后再解引用，而不是定义一个指针指来指去，下边的代码综合和柔性数组会遇到这种情况。

示例代码1

以下示例代码包含了上边讨论的诸多情况，有助于对上述讨论内容的理解

#include <stdio.h>

int a[10] = {0};
int main()
{
    int i;
    for (i = 0; i < 10; i++)
    {
        a[i] = i + 1;
    }

    int *p = a;
    int *p1 = &a[0];
    int *p2 = &a[1];
    int(*p_half)[5] = (int(*)[5]) & a;
    int(*p_2)[2] = (int(*)[2]) & a[1];

    printf(" p     = %p\n", a);
    printf(" p1    = %p\n", p1);
    printf(" p_half = %p\n", p_half);

    printf(" p2    = %p\n", p2);
    printf(" p_2   = %p\n", p_2);

    printf("%d, %d, %d\n", p[0], p1[0], *((int *)p_half));

    printf("%d , %d\n", a[5], *((int *)(p_half + 1)));

    printf("%d , %d\n", a[3], *((int *)(p_2 + 1)));
}

&在结构体中的应用

结构和数组的联系与区别

数组和结构相似的地方是都不再是单一的原子类型，而是一个集合。区别在于数组是相同类型元素（元素可以是原子类型的整型、浮点型、字符型或者结构体、数组等）的集合，而结构是不同类型元素（元素可以是原子类型的整型、浮点型、字符型或者结构体、数组等）的集合。所以结构无所不包，当然结构中也可以定义相同类型的元素，而数组则不然，从这个角度，结构体的普世意义更为广泛，但是毫无疑问，如果结构定义为成员类型是相同的，那么相比定义数组要麻烦的多。

(来自 https://blog.csdn.net/zhanshen112/article/details/80429861 )

为更为彻底的体会&的妙用，我们的示例代码将涉及柔性数组(也叫零长度数组)

示例代码2

#include <stdio.h>
#include <stdlib.h>

struct test
{
        int count;
        //reverse is array name;the array is no item;
        //the array address follow test struct
        int reverse[0];
};

int main()
{
        int i;
        struct test *ptest = (struct test *)malloc(sizeof(struct test) + sizeof(int) * 10);
        for (i = 0; i < 10; i++)
        {
                ptest->reverse[i] = i + 1;
        }
        for (i = 0; i < 10; i++)
        {
                printf("reverse[%d]=%d \n", i, ptest->reverse[i]);
        }
        printf("\n\n");

        printf("sizeof(struct test) =%ld\n", sizeof(struct test));

        /* a+i 或 &a[0]+i = 被取址变量本身的地址  +  i * sizeof(被取址的变量的类型)*/
        int a = *(&ptest->count + 1);
        printf("&ptest->count    = %p\n", &ptest->count);
        printf("&ptest->count +1 = %p\n", (&ptest->count + 1));
        printf("a=%d\n\n\n", a);

        int(*p)[5] = (int(*)[5])(&ptest->reverse[0]);

        // int(*p)[5] = (&ptest->reverse[0]);
        /*
        &ptest->reverse[0]<=>数组0号元素的地址，类型为int* ，所以此处编译器会给出警告
        */

        /* 无论&取哪个成员的地址，只要加上正确的偏移，总能正确访问到目标 */
        printf("%d ,%d\n", *(&ptest->count + 1), *((int *)p));
        printf("%d ,%d\n", *(&ptest->count + 1 + 5), *((int *)(p + 1)));
        printf("%d ,%d\n", *(&ptest->reverse[0] + 5), *((int *)(p + 1)));
        printf("%d ,%d\n", *(&ptest->reverse[1] + 4), *((int *)(p + 1)));

        free(ptest);
        return 0;
}
